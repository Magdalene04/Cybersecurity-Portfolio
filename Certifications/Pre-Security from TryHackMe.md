## 🧠 Key Domains & Hands-on Lab Mastery

### 1. Cyber Security Introduction
* **What I Did:** Explored the core concepts of offensive (Red Team) and defensive (Blue Team) security. 
* **Key Takeaway:** Learned how to approach infrastructure with an adversarial mindset while understanding the defensive countermeasures required to protect it.

### 2. Network Fundamentals
* **What I Did:** Dived deep into how traffic moves across the internet. I analyzed the OSI Model, packets, and fundamental protocols (HTTP, DNS, DHCP, ARP).
* **Practical Application:** Used command-line utilities like `ping`, `traceroute`, and network scanning basics to map out network hops.

### 3. How the Web Works
* **What I Did:** Studied the architecture of web applications, focusing on requests/responses, status codes, and how browsers interpret data.
* **Practical Application:** Examined how web servers handle authentication and sessions, laying the groundwork for analyzing JWT and cookie vulnerabilities.

### 4. Linux & Operating Systems Fundamentals
* **What I Did:** Spent extensive time in the terminal learning file system navigation, file permissions (chmod), and user privileges across OS architectures.
* **Practical Application:** Practiced utilizing commands like `pwd`, `cd`, `mkdir`, `ls`, `touch`, `find`, `curl`, and `ssh` to interact with target systems and analyze local configuration files.

### 5. Computer & Software Fundamentals
* **What I Did:** Studied how computers process data beneath the interface, focusing on how hardware components interact with software applications and data structures.
* **Key Takeaway:** Gained a structural understanding of application architecture, which is vital for identifying how malicious inputs can disrupt normal software execution.

### 6. Software Basics and Attacks & Defences
* **What I Did:** Analyzed common vulnerability classes and weaponization vectors, examining how systems are exposed and how defensive security measures mitigate these exploits.
* **Practical Application:** Explored core defense mechanisms, log review concepts, and proactive security practices used to shield software from unauthorized access.
